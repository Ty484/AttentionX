# AttentionX - Automated Content Repurposing Engine

## 🚀 Problem Statement
Long videos like lectures and podcasts are hard to consume in today’s short-form content world.

## 💡 Solution
AttentionX automatically converts long videos into short viral clips using AI + video processing.

## ✨ Features
- Upload long videos
- Detect important moments automatically
- Extract short clips using FFmpeg
- Convert to vertical format (Reels/TikTok)
- Generate captions (auto subtitles)

## 🛠️ Tech Stack
- Python
- Streamlit
- FFmpeg
- OpenCV
- MediaPipe

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
